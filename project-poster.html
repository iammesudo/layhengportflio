AI System Builder

Poster Generation Tool

An AI-driven platform built for a freelance marketing agent — it transforms raw client inputs into cohesive brand identities and production-ready marketing posters using generative AI and image-to-image synthesis.

Role
AI System Builder
Type
AI Automation Tool
Industry
Marketing / Creative
Client
Freelance Agency
Year
2024
The Problem
"A freelance marketing agent was spending hours manually creating poster variations for clients — same concept, many sizes, many brand colours, over and over."

The goal was to build a system where the agent could input key brand parameters and let AI generate a full suite of visual assets — dramatically cutting production time per client.

My Contribution

I designed and built the full AI pipeline — connecting image generation models, building the input interface, and creating the workflow that turned client briefs into finished assets.

01
AI pipeline design — connected generative image models with structured input processing
02
Image-to-image synthesis — configured style transfer to maintain brand consistency across outputs
03
Input interface — built a no-code form layer for the agent to submit brand briefs without technical knowledge
04
Batch output — automated generation of multiple poster variants per client brief in one run
Outcome

The tool reduced poster production time from hours to minutes. The agent was able to take on more clients without proportionally increasing their workload — a direct business impact through AI.
